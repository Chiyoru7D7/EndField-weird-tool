# InvisibleOp v1.1 — 角色与 Boss 隐身模组

[English](README.md) | 中文版

基于 EFMI（3dmigoto）的明日方舟：终末地 shader-hunting 模组。通过跳过 GPU 绘制调用来隐藏干员、Boss、图标和特效。131 个 hunted shader hash，覆盖 6 个类别。

> **当前适配：** 仅测试过干员 **洛茜 (Rossi)** 和 Boss **罗丹 (Rhodagn)**。其他角色可能未完全覆盖。欢迎贡献更多角色 hash。
>
> **⚠️ 已知问题：** 131 个 hash 中有 65 个（全部 VFX 类别）通过帧分析 diff 自动导入，未经手动狩猎验证。其中部分 hash **与场景物体共享**（粒子、地形、道具），可能导致非干员/Boss 物体被意外隐藏——尤其是环境特效、场景元素、非目标 NPC。如遇此问题，请在 `mod.ini` 中找到并删除对应的 hash。

## 原理

EFMI 拦截 DirectX 绘制调用。当 shader hash 匹配时，`handling = skip` 告诉 GPU："不要画这个物体。"每个角色由多种材质组成——身体、头发、武器、影子、特效——每种都需要各自的 hash。

## 快捷键

| 按键 | 功能 |
|------|------|
| **F1** | 总开关 — 隐藏全部（干员、Boss、图标、特效） |
| **F2** | 仅隐藏干员（身体 + 武器 + 影子 + 特效） |
| **F3** | 仅隐藏 Boss（身体 + 影子 + 技能特效） |
| **F10** | 重新加载所有 mod |

## Shader Hunting 指南

### 按键

| 按键 | 功能 |
|------|------|
| **Home** | 进入/退出狩猎模式 |
| **]** / **[** | 切换像素着色器（PS） |
| **\\** | 标记 PS hash → 剪贴板 |
| **F6** / **F5** | 切换顶点着色器（VS） |
| **F7** | 标记 VS hash → 剪贴板 |
| **Ctrl+]** / **Ctrl+[** | 切换索引缓冲 |
| **End** | 切换标记模式 |
| **Insert** | 结束狩猎（恢复全部隐藏） |
| **Shift+F11** | 帧分析导出（整个帧） |

### 狩猎步骤

1. 按 **Home** — 左上出现绿色覆盖层
2. 按 **]** 循环 PS hash，观察目标
3. 目标闪烁/消失 → 按 **\\** 复制 hash
4. 粘贴 hash 到 `mod.ini` 对应段落
5. F10 重新加载，测试

### PS vs VS：用哪个？

| | 像素着色器 (PS) | 顶点着色器 (VS) |
|---|---|---|
| **何时使用** | 默认选择，90%+ 情况有效 | PS 全部试过后仍可见 |
| **狩猎方式** | ] / [ | F6 / F5 |
| **INI 设置** | `Hash = ...`（默认） | `Hash = ...` + `match_type = vs` |
| **风险** | 低。一个 shader = 一种材质 | 较高。可能隐藏非目标物体 |

### 帧分析：捕获瞬时特效

技能施放、短暂特效和影子 shader 出现太快，手动狩猎难以捕获：

1. **待机帧**：Boss/角色待机 → 按 **Shift+F11**
2. **技能帧**：技能/特效播放中 → 按 **Shift+F11**
3. 运行 diff 工具：
   ```
   powershell .\diff_frames.ps1
   ```
4. 脚本输出仅在技能帧出现的新 hash，格式可直接粘贴到 mod.ini
5. 加入 mod.ini，F10 重新加载，测试

### 实用技巧

- 狩猎前**关闭所有其他 mod**——其他 mod 的覆盖层会干扰
- **在大世界狩猎**，不要用角色菜单（LOD 变体不同）
- **走远一些**触发 LOD 切换，顺便狩猎低细节版本
- **用你平时玩的画质设置**——高/中/低预设使用不同的 shader 变体
- **在多个地图测试**——部分 shader 是分地图加载的
- **立即粘贴 hash**——alt-tab 切换窗口后覆盖层消失

## 安装

1. 复制 `InvisibleOp` 文件夹到：
   ```
   <EFMI 目录>\Mods\InvisibleOp\
   ```
   通常路径：`%APPDATA%\XXMI Launcher\EFMI\Mods\InvisibleOp\`
2. 通过 XXMI Launcher 启动游戏（确保 EFMI 已启用）
3. 游戏内按 **F10** 加载

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| Mod 不加载（F10 无效） | 检查 XXMI Launcher → EFMI 已启用 |
| 角色部分可见 | 漏了 shader。分别狩猎脸/眼睛/发梢。尝试 VS（F6） |
| 漂浮的眼睛/牙齿 | 脸部 shader 在循环中靠后出现——继续按 ] |
| 环境物体消失 | 删除该 hash。范围太广。尝试 VS 匹配 |
| F1 不能隐藏全部 | 检查 mod.ini → [Constants] 与下一段之间有空行 |
| 狩猎覆盖层无显示 | 镜头对准角色。持续无果则重启游戏 |
| F10 后游戏崩溃 | mod.ini 语法错误——检查 `endif` 缺失、段名重复 |
| 游戏更新后 mod 失效 | Shader hash 已变更。重新狩猎 |
| VS 切换无效（Shift+[ ]） | 改用 F5/F6/F7——已在 d3dx.ini 中修改 |

## 封号风险

本 mod 使用 EFMI 运行时注入。明日方舟：终末地使用 **ACE 内核级反作弊**。**存在真实封号风险。**

- 切勿在主账号使用
- 切勿发布与账号关联的 mod 截图
